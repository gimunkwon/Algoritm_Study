# 백준 1753 - 최단경로 (골드 5)

문제 링크: https://www.acmicpc.net/problem/1753

## 문제 요약

방향 그래프가 주어지면, 주어진 시작점에서 다른 모든 정점으로의 최단 경로 값을 구한다.

- 그래프는 **방향 그래프**, 같은 두 정점 사이에 여러 간선 존재 가능
- 간선 가중치는 10 이하의 자연수
- 도달 불가능한 정점은 `INF` 출력

## 풀이 접근

시작 정점 K를 출발점으로 **다익스트라**를 수행해 모든 정점까지의 최단 거리를 구한다.  
방향 그래프이므로 간선을 단방향으로만 추가한다.

## 핵심 포인트

**우선순위 큐에 `{비용, 노드}` 순서로 넣어야 min-heap이 비용 기준으로 정렬된다.**

```cpp
priority_queue<pil, vector<pil>, greater<pil>> pq;
pq.push({distance[startnode], startnode});  // {비용, 노드} 순서
```

`pair`는 first 기준으로 먼저 정렬되므로, first에 비용을 넣어야 최소 비용 노드가 먼저 꺼인다.

**지연 삭제(lazy deletion)로 중복 처리 방지**

```cpp
if (cost > distance[node]) continue;  // 이미 더 짧은 경로로 처리된 경우 스킵
```

우선순위 큐에 같은 노드가 여러 번 들어갈 수 있으므로, 꺼낼 때 현재 최단 거리보다 크면 스킵한다.

## 풀이 코드

소스코드: `BJ_1753.cpp`

```cpp
#include <iostream>
#include <vector>
#include <queue>
using namespace std;

const int INF = 1e9;
using pil = pair<int, int>;

vector<int> Dijkstra(int startnode, int V, vector<vector<pil>>& graph)
{
    vector<int> distance(V + 1, INF);
    distance[startnode] = 0;

    priority_queue<pil, vector<pil>, greater<pil>> pq;
    pq.push({0, startnode});

    while (!pq.empty())
    {
        auto top = pq.top();
        int cost = top.first, cur = top.second;
        pq.pop();

        if (cost > distance[cur]) continue;

        for (const auto& edge : graph[cur])
        {
            int next_node = edge.first;
            int weight = edge.second;

            if (cost + weight < distance[next_node])
            {
                distance[next_node] = cost + weight;
                pq.push({distance[next_node], next_node});
            }
        }
    }

    return distance;
}

int main()
{
    ios_base::sync_with_stdio(false);
    cin.tie(NULL);

    int V, E;
    cin >> V >> E;

    int start;
    cin >> start;

    vector<vector<pil>> graph(V + 1);

    for (int i = 0; i < E; i++)
    {
        int u, v, w;
        cin >> u >> v >> w;
        graph[u].push_back({v, w});  // 방향 그래프: 단방향 추가
    }

    vector<int> distance = Dijkstra(start, V, graph);

    for (int i = 1; i <= V; i++)
    {
        if (distance[i] == INF) cout << "INF\n";
        else cout << distance[i] << "\n";
    }

    return 0;
}
```

## 실수 포인트 정리

| 실수 | 원인 | 해결 |
|------|------|------|
| 파라미터 shadowing | 함수 파라미터 `node`와 루프 내 지역변수 `node` 이름 충돌 | 루프 내 변수명을 `cur`로 변경 |
| pq push 초기값 | `pq.push({distance[startnode], startnode})`로 작성 시 명시성 낮음 | `pq.push({0, startnode})`로 명시 |
