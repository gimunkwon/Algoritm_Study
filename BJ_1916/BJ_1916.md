# 백준 1916 - 최소비용 구하기 (Silver 1)

문제 링크: https://www.acmicpc.net/problem/1916

## 문제 요약

N개의 도시와 M개의 단방향 버스 노선이 주어진다.  
A번 도시에서 B번 도시까지 가는 데 드는 **최소 버스 비용**을 구한다.

- 그래프는 **단방향**, 버스 비용은 0 이상 100,000 미만의 정수
- 항상 시작점에서 도착점으로의 경로가 존재

## 풀이 접근

시작 도시 A를 출발점으로 **다익스트라**를 수행해 모든 도시까지의 최단 거리를 구한다.  
BJ_1753(1→전체)과 달리 A→B 단일 경로의 최솟값만 출력한다.

## 핵심 포인트

**pq에는 누적 총 비용을 넣어야 한다.**

```cpp
if (weight + cost < distance[node])
{
    distance[node] = weight + cost;
    pq.push({weight + cost, node});  // 엣지 가중치(weight)만 넣으면 안 됨
}
```

`weight`는 현재 간선의 가중치일 뿐이고, 이전까지 온 누적 비용 `cost`를 더한 값을 넣어야 한다.  
`weight`만 넣으면 pq의 정렬 순서가 틀어져 잘못된 경로를 먼저 처리할 수 있다.

**지연 삭제(lazy deletion)로 중복 처리 방지**

```cpp
if (cost > distance[next_node]) continue;
```

pq에 같은 노드가 여러 번 들어갈 수 있으므로, 꺼낼 때 저장된 최단 거리보다 크면 스킵한다.

## 풀이 코드

소스코드: `BJ_1916.cpp`

```cpp
#include <iostream>
#include <vector>
#include <queue>
using namespace std;

const int INF = 1e9;
using pil = pair<int, int>;

int Dijkstra(vector<vector<pil>>& graph, int startnode, int endnode)
{
    vector<int> distance(graph.size(), INF);
    distance[startnode] = 0;

    priority_queue<pil, vector<pil>, greater<pil>> pq;
    pq.push({0, startnode});

    while (!pq.empty())
    {
        auto top = pq.top();
        int cost = top.first, next_node = top.second;
        pq.pop();

        if (cost > distance[next_node]) continue;

        for (auto i : graph[next_node])
        {
            int weight = i.first, node = i.second;

            if (weight + cost < distance[node])
            {
                distance[node] = weight + cost;
                pq.push({weight + cost, node});
            }
        }
    }

    return distance[endnode];
}

int main()
{
    ios_base::sync_with_stdio(false);
    cin.tie(NULL);

    int N, M;
    cin >> N >> M;

    vector<vector<pil>> graph(N + 1);

    for (int i = 0; i < M; i++)
    {
        int start, end, cost;
        cin >> start >> end >> cost;
        graph[start].push_back({cost, end});
    }

    int start_city, end_city;
    cin >> start_city >> end_city;

    cout << Dijkstra(graph, start_city, end_city);

    return 0;
}
```

## 실수 포인트 정리

| 실수 | 원인 | 해결 |
|------|------|------|
| pq에 엣지 가중치만 삽입 | `pq.push({weight, node})`로 누적 비용 누락 | `pq.push({weight + cost, node})`로 수정 |
