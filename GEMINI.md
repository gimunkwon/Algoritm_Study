- **개념 설명 및 C++ 개발 (Claude 전담)**: 알고리즘 설명 작성, C++ 소스 코드 구현(`.cpp`), 그리고 업로드용 마크다운 파일(`.md`) 생성 등 개발과 관련된 작업은 Claude가 담당합니다.

## 💻 C++ 프로젝트 개발 및 브랜치 전략 (CLAUDE.md 연계)

1. **Git 브랜치 관리**: 새로운 알고리즘 주제를 시작할 때 해당 주제명으로 브랜치를 생성하여 작업합니다 (예: `Dijkstra`, `BFS`, `DP`).
2. **VS 프로젝트 구조**: `AlgoritmStudy.sln` 솔루션 파일에 새 프로젝트(`주제명_01.vcxproj`)를 등록하여 사용합니다.
3. **단일 소스 파일 원칙**: 각 주제별 프로젝트 폴더 아래 하나의 `.cpp` 파일 안에 관련 개념 설명(주석)과 실제 컴파일 가능한 예제 코드를 함께 포함합니다.
4. **컴파일 설정**: 프리컴파일드 헤더는 사용하지 않습니다 (`PrecompiledHeader=NotUsing`).
